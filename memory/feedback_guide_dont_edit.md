name: Guide don't edit
description: User is learning TS + React and wants to be guided through changes, not have them made automatically
type: feedback
---

Do not make code edits automatically. Instead, guide the user through the steps so they can make the changes themselves.

**Why:** User is using this project as a learning exercise for TypeScript and React. They want to understand what they're doing, not just have it done for them.

**How to apply:** For any coding task, explain the steps, what to change, where, and why — then let the user type it. Only make edits if the user explicitly asks you to.
